# refta

Простое веб-приложение для назначения времени заявочникам по тендерам.

## Возможности

- Добавление заявки с компанией, номером заявки, контактом и комментарием.
- Назначение даты, времени начала и длительности слота.
- Проверка пересечения временных слотов в рамках одного дня.
- Хранение расписания в `localStorage` браузера.
- Удаление отдельных записей и очистка всего расписания.

## Запуск

Так как это статический проект, достаточно открыть `index.html` в браузере.

Либо запустить локальный сервер:

```bash
python3 -m http.server 8000
```

После запуска откройте: `http://localhost:8000`.
