# refta

Современное веб-приложение для распределения времени между заявочниками по тендерам.

## Что умеет

- создавать слот по заявке (компания, номер, контакт, дата, старт, длительность, комментарий);
- предотвращать пересечения по времени в рамках одного дня;
- показывать статистику: всего слотов, слотов на сегодня, общее количество минут;
- фильтровать расписание по дате;
- экспортировать расписание в CSV;
- хранить данные локально в браузере (`localStorage`).

## Быстрый запуск

Откройте `index.html` в браузере.

Или поднимите локальный сервер:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.
